6.5 🔥 凤凰产品开发逻辑 = 人群 × 场景 × 品类 三维交集

核心原则：差异化 ≠ 参数堆叠 · 差异化 = 找到"人群 × 场景 × 品类"三维交集的空白位置

"比 Pacific 轻 1kg" / "比 Element 多 2 档变速" → 同质化 = 价格战

"给公路玩家在雅加达外环周末赛的UCI 公路，跳过 5-9.5M 太平洋主力 → 12-15M 真空段 + UCI 情绪 + FNIX SNI 错位"
🎯 凤凰产品决策树（V2.6.1 战略 → V2.9.3 升级）
  1. 人群定位：选目标人群 (公路玩家 + 平把通勤族 + 长途玩家) → 不要先想"我要做什么车"

  3. 品类聚焦：匹配品类 (公路 / Hybrid / Gravel) → 跳过 MTB / 折叠 / 童车 永久不参与
  4. Pacific 防火墙：跳过 5-9.5M 太平洋主力价位 → 切入 12M+ 中高端真空
  5. 功能价值：在选定品类里选 中高端价段 (12-18M 公路 / 7-15M Hybrid) → 避开参数红海
