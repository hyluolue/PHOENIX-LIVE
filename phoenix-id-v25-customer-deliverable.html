🎯 双品牌战略 · 永不和 Pacific 直接竞争

V2.6 关键修订：折叠车品类战略降级

· 降级原因：Phoenix 无折叠核心技术（中国头部 MTB/公路车品牌，不是折叠车品牌）
· 强竞品：大行折叠 + JAVA 折叠 = 用户心智已占 = Phoenix 不可能快速建立心智
· 不可形成复利：折叠车技术迭代慢 + 用户教育成本高 + 流量入口被竞品锁 = 长期 ROI 低
· 新策略：仅 1 SKU 试水（看市场反应） → 如客户坚持 → 不投入长期资源

• 定位：全球背书 · 国际中端骑行品牌
• 核心认证：UCI（国际自行车联盟）
• 成本优势：Walmart 规模（成本控制强）
• 价位：8-12M IDR 中端主流
• 品类：折叠 + 主流公路车
• 目标：Top 5（ID 中端段第一）

FNIX 品牌身份

• 定位：印尼本土品质标准品牌（全新）
• 核心认证：SNI 印尼国家标准
• 成本优势：本土供应链（短链 + 本土制造）
• 价位：10-20M IDR 中高端真空
• 品类：高端砾石 + 高端 MTB
• 目标：Top 3（ID 中高端段第一）

PACIFIC 客户自有品牌 · 错位铁律

• Pacific 主流：5.1-9.5M IDR 入门通勤（印尼市占率第一·客户亲儿子）
• ❌ Phoenix 绝不能进 5-10M（会抢 Pacific 主战场）
• ❌ FNIX 绝不能抢 Pacific 入门通勤用户
• ✅ Phoenix/FNIX 只做 Pacific 没优势的品类（折叠 / 高端砾石 / 25M+ 旗舰）
