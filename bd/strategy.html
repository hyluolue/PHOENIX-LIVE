3+1 公式：1 人群 + 1 场景 + 1 竞品 + 1 解决方案

目的：基于现状 4 维度做战略选择

⏰ 8.6 时间场景差异化法 (V2.9.23 V3+V4 共同点提炼)

· 差异点 = V3 防锈 / V4 夜光 = 不冲突
· 整合价值 = 24 小时全场景守护 = 凤凰品牌核心
📋 时间场景差异化 3 步：
① 列出目标市场的所有真实使用时间（早/中/晚 + 雨季/旱季 + 工作日/周末）
② 找出每个时间段下的真实痛点（雨季生锈 / 夜间看不见）
③ 把不同时间段痛点 = 不同子场景 = 整合为 1 个品牌承诺（24h 全场景守护）
→ 3 个子场景 < 1 个品牌承诺 = 战略协同 ≠ 战略分散

⚙️ 8.9.1 V1 质价比（不是性价比）—— 凤凰规模优势 + 共享单车供应商 + 上市公司

· 战略含义：竞品（Veloce / Marine / Foxter 等）大多没有自己的实验室 → 凤凰可以拿 "我实验室测过的" 当营销话术

🛡️ 8.9.3 V3 耐久度（不是耐候性）—— 消费者心智认知 = 真实护城河
