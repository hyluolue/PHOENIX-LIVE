🧠5 大理论框架中心 跨 Tab 决策参考

📌 核心原则：所有 Tab 的"决策建议 / 洞察 / 推荐"都以这 5 大理论为参考依据。
📌 5 看 = 埃森哲 · 4P = 极简市场营销 · 9 维 = 产品定义画布 · 3+1 = 爆品方法论 · SWOT = 经典营销

1️⃣ 埃森哲五看方法论

核心：市场洞察 = 5 维度信息输入
方法：看宏观 (PEST) + 看行业 + 看客户 + 看竞争 + 看自己
使用：每个 Tab 对应一"看"· 5 看整合 = 总览

2️⃣ 极简市场营销 SOSTAC+STP+4P

SOSTAC：Situation→Objectives→Strategy→Tactics→Actions→Control
STP：市场细分 (4 维) → 目标客户 → 品牌定位
4P + 4C：Product/Price/Place/Promotion ↔ 客户视角
使用：看行业 (Situation) + 看客户 (STP) + 看自己 (4P)

3️⃣ 产品定义画布 9 维 + 6 步工作流

产品侧 7 项：客户细分 / 应用场景 / 竞品分析 / 价值主张 / 功能设定 / 关键指标 / 重要支持
市场侧 2 项：成本分析 + 收益分析
6 步：研究 → 机会 → 概念 → 可行性 → 方案 → 资源分配
使用：看自己 6.1-6.6 (产品开发逻辑)

4️⃣ 2024 爆品设计方法论 (周立钢)

3+1 公式：1 人群 + 1 场景 + 1 竞品 + 1 解决方案
3 大价值：功能 (痛) + 情绪 (爽) + 资产 (痒)
FABE：特征-优点-利益-证据 · 4 象限：机会×能力
使用：看自己 8.1-8.6 (爆品方法论)

5️⃣ SWOT 4 象限 (Phoenix 4 维度现状)

SWOT：优势 (S) / 弱势 (W) / 机遇 (O) / 威胁 (T)
目的：基于现状 4 维度做战略选择
用法：S+O = 增长战略 · W+O = 转型战略 · S+T = 多元化 · W+T = 防御
使用：看自己 9. SWOT 4 象限

🇨🇳 6.4 印尼华人圈战略 (Phoenix 顶层战略机会)

🇮🇩 印尼华人规模
1000+ 万 · 全球第一
印尼华人占全球海外华人 40% · 集中雅加达/泗水/棉兰
🇲🇾 马来西亚华人
700+ 万 · 全球第二
马来华人占全国 22.6% · 文化保留度更高
💎 核心洞察："顺中国国力强大 = 顺欧美品牌出海成功逻辑" · 100 年前欧美国力强 → 欧美品牌（奔驰/宝马/可口可乐）出海成名牌 → 100 年后中国国力强 → 中国品牌（华为/比亚迪/凤凰）出海 = 顶级机会
华人圈策略范围：🇮🇩 ID（1000+ 万）= 华人圈 P0 · 🇲🇾 MY（700+ 万）= 华人圈 P2 · 🇧🇩 BD = 纯本地穆斯林文化融合（不做华人圈）
V5 双圈层战略：对华人圈 = "民族品牌文化共鸣"（凤凰 = 中华百年民族品牌）· 对本地穆斯林圈 = "本地文化融合（蜡染 batik）" = 1000+ 万华人 + 2.7 亿本地 = 3.8 亿潜在市场
